**Description:**

**Acceptance Criteria:**

* Reasoning discipline is scored separately from image accuracy.

## Epic 8 — Contradiction Detection and Resolution

### Task 8.1 — Detect material contradictions

**Description:**  
Report pairs are compared for each canonical claim.

Pairs:

* R1 vs R2
* R1 vs R3
* R2 vs R3

**Acceptance Criteria:**

* W7 Contradiction Matrix is produced.
* Each contradiction has ID, claim, reports, statements, EvidencePointers, modality pair, type, impact, and resolution.
* Materiality threshold is applied.
* Bias flips receive high priority.

### Task 8.2 — Resolve contradiction types

**Description:**  
Contradictions are resolved using framework rules.

Resolution types:

* Type A: Text contradiction
* Type B: Chart interpretation divergence
* Type C: Mixed support

**Acceptance Criteria:**

* Type A favors stronger traceability.
* Type B favors reasoning discipline, not chart truth.
* Type C favors text-verifiable support for factual resolution.
* Resolution notes remain concise and objective.

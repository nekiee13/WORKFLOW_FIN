## Task 6.2 — Compute Deterministic Scores

**Goal:** Score each report using the fixed rubric.

**Actions**

* Use section maximums:
  * A = 35
  * B = 25
  * C = 25
  * D = 15
* Apply formula:

```text
section_score = MAX(section_max - Σ(error_count × deduction), 0.2 × section_max)
weighted_total = A + B + C + D
```

* Assign grades:
  * Exceptional: ≥90%
  * Strong: 80–89%
  * Adequate: 70–79%
  * Weak: 60–69%
  * Poor: 50–59%
  * Failing: <50%

**Acceptance Criteria**

* W3 Detailed Scorecards exist for R1–R3.
* W3B Final Model Scorecard Overview exists.
* Scores are reproducible from issue counts.
* Section floors are respected.
* Weighted totals sum from section scores.

# Execution Guardrails

## Narrative Guardrails

* Objective narration will be maintained.
* Third-person perspective will be used.
* Personal pronouns will be avoided.
* Sentences will remain short where possible.
* Claims will be written in beginner-friendly language.

## Evidence Guardrails

* Every non-user claim will receive an EvidencePointer.
* User-supplied values will be treated as ground truth.
* No recomputation will occur without a stated formula.
* Chart absence will not reduce scores.
* Chart-dependent claims will be scored only for reasoning discipline.

## Scoring Guardrails

* Deductions will follow the deterministic table.
* Section floors will be enforced.
* Self-audit credit will be applied only when allowed.
* Numerical exactness will remain secondary, except where internal inconsistency indicates reasoning failure.

## Contradiction Guardrails

* Materiality thresholds will be applied before logging contradictions.
* Text-verifiable claims will control factual resolution when support is mixed.
* Chart interpretation disputes will not be resolved as visual fact.
* Stronger traceability will control text contradiction resolution.
